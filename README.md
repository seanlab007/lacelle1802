## Français

### Présentation de la Maison

LA CELLE PARIS est une prestigieuse maison de parfum française fondée en 1802 par Léa Celle dans les champs de fleurs de Grasse. Depuis plus de deux siècles, notre maison perpétue l'art sacré de la parfumerie française au service des plus grandes cours d'Europe.

### Site Officiel

🌐 **https://www.lacelle1802.com**

### Notre Histoire

En 1802, dans les champs de lavande et de roses de Grasse, Léa Celle crée sa première fragrance. Son art olfactif conquiert rapidement les palais les plus exigeants d'Europe. Aujourd'hui, LA CELLE perpétue cette tradition d'excellence avec des créations qui allient tradition et modernité.

## Development

### Tech Stack

- **Framework**: React + TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Package Manager**: pnpm

### Getting Started

```bash
# Install dependencies
pnpm install

# Development server
pnpm dev

# Production build
pnpm build

# Preview production build
pnpm preview
```

### Deployment

The site is deployed on [Render](https://render.com). Configuration can be found in `render.yaml`.
